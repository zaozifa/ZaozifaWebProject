具体字页中间标题，如果有多个字，则第一个多为正体字，后面是简体或异体字或错体字

此造字法暂仅仅释说文解字9000+汉字，其他汉字不录，个人以为其他汉字无造字之法，随意拼凑，不必存在，更不必学，如武则天所造，如营销汉字biangbiang等

个人之见，所有汉字字典，说文解字最好

个人之见，通假字是错误说法，不存在通假字，读不懂也不要硬说它是通假，很简单不管它，或牵挂于心

个人之见，一字一音，不存在多音字

个人之见，尚书语法自由，长则长，不刻意四字

个人之见，普通话多复词，句式甚长冗杂难懂，不如各地方言，方言绝少复词，此所谓复词即两字同意，复词之法或自日本
